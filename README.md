# Rock Paper Scissors Game 

A fun and interactive **Rock Paper Scissors game** created using **HTML, CSS, and JavaScript**.  
This project helped me practice JavaScript logic, random functions, and user interaction.

##  Features
- Computer choice using `Math.random()`
- Real-time score tracking
- Clean and responsive UI
- Instant result display (Win/Lose/Draw)

##  What I Learned
- Using random logic in JavaScript
- DOM manipulation and event handling
- Handling UI updates dynamically
- Writing clean and modular JS code
